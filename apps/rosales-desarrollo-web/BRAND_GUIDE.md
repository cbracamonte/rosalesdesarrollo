## Tipografía

### Familia de Fuentes

#### Para Títulos y Encabezados
- **Poppins Bold** (peso 700)

**Tamaños:**
- Título principal (H1): 48px - 72px
- Subtítulos (H2): 32px - 40px
- Secciones (H3): 24px - 28px

#### Para Cuerpo de Texto
- **Montserrat Regular** (peso 400)

**Tamaños:**
- Párrafos: 16px
- Texto pequeño: 14px
- Labels: 12px

#### Para Taglines y Descripciones
- **Poppins Medium** (peso 500)

**Tamaño:** 18px - 20px

### Pesos de Fuente
- **Light:** 300 (acentos, subtexto)
- **Regular:** 400 (cuerpo principal)
- **Medium:** 500 (énfasis moderado)
- **Bold:** 700 (títulos y CTA)
- **Extra Bold:** 800 (énfasis máximo)

### Espaciado de Líneas
- Títulos: 1.2x del tamaño de fuente
- Cuerpo: 1.5x - 1.6x del tamaño de fuente
- Pequeño: 1.4x del tamaño de fuente

### Espaciado de Letras (Letter Spacing)
- Títulos: -0.5px a 0px
- Cuerpo: 0px a 0.5px
- Taglines: -0.5px

## Especificaciones de Diseño

### Espaciado General
```
Margen exterior:    40px - 60px
Padding interno:    20px - 40px
Espacio entre elementos: 20px - 30px
Líneas blancas:     40px - 60px
```

### Bordes y Sombras
```
Radio de bordes:    4px - 8px (elementos moderados)
Sombra sutil:       0 2px 8px rgba(42, 45, 55, 0.1)
Sombra media:       0 4px 12px rgba(42, 45, 55, 0.15)
```

### Degradados Sugeridos
```
Fondo web (radial): #1E2129 (centro) → #141720 (bordes)
Oscuro a Plata:     #1E2129 → #C0C5CB
Oscuro a Gris:      #2A2D37 → #9DA2A8
Para fondos claros: #F8F9FA (blanco casi puro)
```

## Directrices de Uso

### Logo Completo
- Ancho mínimo: 200px
- Espacio de respiro: 20px alrededor
- Nunca distorsionar aspecto

### Isotipo
- Tamaño mínimo: 16px (favicon)
- Tamaño recomendado: 80px - 200px
- Siempre mantener aspecto 1:1.14 (aproximadamente)

### Colores en Interfaz
- Texto sobre blanco: #3B3E44 (contraste AAA)
- Texto sobre oscuro: #FFFFFF o #C0C5CB
- Texto sobre gris medio: #2A2D37 o #FFFFFF
- Botones primarios: #2A2D37 con texto #FFFFFF
- Botones secundarios: #FFFFFF con borde #3B3E44
